# Design Brainstorm — Tactical Training Course 3D Viewer

## Three Stylistic Approaches

### 1. Tactical Command Console
A defense-grade operations interface: dark slate cockpit UI, monospace data readouts, amber/green tactical accents, blueprint-grid 3D ground. Feels like a mission planning system.
**Probability: 0.07**

### 2. Architectural Blueprint Studio
Clean light "drafting table" aesthetic, technical line work, schematic models on a white/graph background, like an interior-design / CAD presentation tool.
**Probability: 0.03**

### 3. Field Briefing Deck
Editorial, presentation-first layout with big confident type and muted military earth tones, optimized for projecting to a room of clients.
**Probability: 0.02**

---

## CHOSEN APPROACH: Tactical Command Console

**Design Movement:** Modern defense/aerospace HMI (human-machine interface) — think mission-planning consoles, drone ground-control stations, crossed with clean Swiss data design.

**Core Principles:**
1. Information clarity first — the 3D scene is the hero, UI is a quiet frame around it.
2. Precision & order — aligned grids, tabular data, consistent measured spacing.
3. Confident restraint — dark neutral canvas so colored 3D models and accents pop.
4. Operational feel — subtle HUD framing, corner ticks, monospace coordinates.

**Color Philosophy:** A deep gunmetal/charcoal canvas conveys seriousness and lets schematic models read clearly (high-stakes police/military context). A single signature **tactical amber** (#f5a623-ish) for primary actions and highlights communicates "active/selected" like a real console. Cool steel grays for structure; a muted tactical green as secondary status color.

**Layout Paradigm:** Full-bleed 3D viewport with a left **obstacle library rail** and a right **inspector/controls panel** — an asymmetric console layout, not centered marketing blocks. Top thin command bar with mode toggles (Course View / Single Obstacle / Layouts).

**Signature Elements:**
- HUD corner ticks framing the viewport.
- Monospace coordinate/label chips on obstacles.
- Blueprint grid floor that fades to the horizon.

**Interaction Philosophy:** Direct manipulation — click an obstacle in the library to focus it; orbit/zoom feels weighty and precise. Selection snaps with amber outline.

**Animation:** Snappy ease-out panel slides (<250ms), smooth camera fly-to focus transitions, gentle idle auto-rotate in single-obstacle mode.

**Typography System:** Display/headings = "Oxanium" or "Saira" (technical, squared); body/UI = "Inter Tight"; data/coordinates = "JetBrains Mono". Headings uppercase with wide tracking for command-bar labels.

**Brand Essence:** A precision course-planning console for tactical training providers pitching to police & military. Personality: precise, authoritative, modern.

**Brand Voice:** Terse, operational. Example lines: "DEPLOY COURSE LAYOUT" / "16 OBSTACLES IN ARSENAL". Ban generic filler.

**Wordmark & Logo:** A bold angular monogram — chevron/arrow forming a stylized course path inside a hexagon. Signature color amber.

**Signature Brand Color:** Tactical amber `#F5A623` on gunmetal `#15181d`.
